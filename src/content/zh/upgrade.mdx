### Nextra v4 更新要点

Nextra v4 现已完成升级，主要更新：
- ⚡️ 显著提升了页面加载速度
- 🎨 优化了主题样式系统
- 📱 改进了移动端适配
- 🛠️ 新增更多实用组件

详细信息请参考 [Nextra v4 官方升级指南](https://the-guild.dev/blog/nextra-4)

### Tailwind CSS v4 更新要点

Tailwind CSS v4 升级完成，核心改进：
- 🚀 全新的 JIT 引擎，编译更快
- 💎 优化的样式系统，体积更小
- 🎯 简化的配置方式
- 📦 更多开箱即用的功能

更多细节请查看 [Tailwind CSS v4 升级指南](https://tailwindcss.com/docs/upgrade-guide)

### 迁移说明

为确保顺利升级，建议：

1. 先在测试环境进行升级验证
2. 检查自定义组件的兼容性
3. 查看控制台是否有警告信息
4. 按照官方迁移指南逐步操作
