# 🔥 Nextra v4 & Tailwind CSS v4

### Nextra v4 迁移指南

Nextra v4 带来了许多改进和新特性，具体更新内容请参考[官方升级指南](https://the-guild.dev/blog/nextra-4).

### Tailwind CSS v4 迁移指南

Tailwind CSS v4 进行了底层优化，完整更新内容请参考[官方升级指南](https://tailwindcss.com/docs/upgrade-guide).

### 其他注意事项

- 由于 `Nextra v4` 和 `Tailwind CSS v4` 都进行了底层优化，建议逐步迁移并在测试环境充分测试。
- 如果遇到问题，可以查阅[官方文档](https://nextra.site)和[Tailwind CSS](https://tailwindcss.com)的迁移指南。

如果有任何问题或建议，请提交 Issue 或 PR！
