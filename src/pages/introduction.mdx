# Introduction

A free, open-source, and powerful Template with Next.js + Nextra + TypeScript + Tailwind CSS.

```js
console.log('hello, world')
```

## 安装和运行

* 安装依赖

```bash
pnpm install
```

* 本地开发

```bash
pnpm run dev
```
