# 简单介绍

一个免费、开源且功能强大的模板，采用 Next.js + Nextra + TypeScript + Tailwind CSS

```js
console.log('hello, world')
```

## 安装和运行

* 安装依赖

```bash
pnpm install
```

* 本地开发

```bash
pnpm run dev
```
